  BOT 3: DeFi Yield Bot — Setup-Anleitung

  Zwei Teile: Bot-Setup + echte Investition

  Teil A — Bot konfigurieren (10 Min):

  Etherscan API Key holen (kostenlos):
  1. https://etherscan.io/register → Account erstellen
  2. https://etherscan.io/myapikey → "Add" → Key kopieren
  3. In .env eintragen: ETHERSCAN_API_KEY=dein-key

  Arbiscan API Key (für Arbitrum):
  1. https://arbiscan.io/register → Account erstellen
  2. https://arbiscan.io/myapikey → Key kopieren
  3. In .env eintragen: ARBISCAN_API_KEY=dein-key

  Teil B — 5000 EUR investieren:

  Ich empfehle Aave V3 auf Arbitrum (günstigste Gas-Gebühren):

  Schritt 1 — EUR zu USDC:
    → Kraken/Coinbase/Bitpanda
    → 5000 EUR kaufen als USDC
    → USDC zu deiner Wallet senden:
      0xF9A5EaE7E0b1D51cc8411BB612bA3B9D2cb4500A

  Schritt 2 — USDC zu Arbitrum bridgen:
    → https://bridge.arbitrum.io
    → USDC von Ethereum → Arbitrum
    (spart 90% Gas-Gebühren beim Compounding!)

  Schritt 3 — Auf Aave einzahlen:
    → https://app.aave.com
    → Netzwerk: Arbitrum
    → "Supply" → USDC → Betrag eingeben → Bestätigen
    → Fertig! Zinsen laufen sofort

  Schritt 4 — Bot übernimmt:
    → 2_DAUERLAUF.bat starten
    → Alle 6h: Yield-Check, Bericht, Alert wenn APY fällt
